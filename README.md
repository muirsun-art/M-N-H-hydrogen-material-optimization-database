# M-N-H-hydrogen-material-optimization-database
本项目利用 大型语言模型（LLM） 结合 精心设计的提示词（prompt engineering），针对不同的储氢体系优化策略，构建了一个涵盖多维度信息的金属-氮-氢储氢材料性能数据库。该数据库旨在为储氢材料研究人员提供系统化、可检索的数据支持，助力研究者深入分析不同合成条件与储氢性能之间的关系，加速新型储氢材料的设计与优化。
This project utilizes Large Language Modeling (LLM) combined with carefully designed prompt engineering to construct a performance database of metal-nitrogen-hydrogen storage materials, encompassing multi-dimensional information, for different hydrogen storage system optimization strategies. This database aims to provide systematic and searchable data support for hydrogen storage material researchers, assisting them in deeply analyzing the relationship between different synthesis conditions and hydrogen storage performance, and accelerating the design and optimization of novel hydrogen storage materials.
